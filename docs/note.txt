[en-us]
PEP-8: It is a well-recognized code style used by the Python community.

Typing tips: It's a way for you to indicate to the developer that you will see your
code in the future, what is the type of an attribute or a variable or a method and even
even what this method can return.

Code Lint: It's a way for you to validate the style of code you are using.

[pt-br]
PEP-8: É um estilo de código bem reconhecido e usado pela comunidade Python.

Type hints: É uma forma de você indicar para o desenvolvedor que vai ver o seu
código no futuro, qual o tipo de um atributo ou de uma variável ou de um método e até
mesmo o que esse método pode retornar.

Code Lint: É uma forma de você validar o estilo de código que você está usando.
